Run Commands — TrackMyExpenses

This file lists the commands to install dependencies and run the application.

1. Install dependencies

npm install

2. Start the server

Default (uses the `start` script defined in package.json):

npm start

Or explicitly with Node:

node server/server.js

3. Development (same as start)

npm run dev

4. Set a custom port (PowerShell)

$env:PORT=4000
npm start

5. Quick verification

curl http://localhost:3000/
Note: Ensure you have Node.js installed on your system to run these commands. You can download it from https://nodejs.org/.
